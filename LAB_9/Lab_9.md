# Wireless Network Setup and Packet Analysis Lab Report

A professional lab report documenting the configuration of a wireless local area network (WLAN) using Cisco Packet Tracer.

## Overview

This lab report demonstrates the setup and analysis of a basic wireless network topology consisting of:
- 1 × WRT300N Wireless Router
- 2 × Laptop-PT devices (wireless enabled)

## Lab Objectives

- Configure a wireless router as an access point and DHCP server
- Connect wireless clients to the network
- Test connectivity between devices
- Analyze packet flow using Cisco Packet Tracer's simulation mode

## Network Configuration

### Topology
The network uses a star topology with the WRT300N wireless router at the center, providing connectivity to two laptops via Wi-Fi.

### IP Addressing Scheme
- **Router**: 192.168.0.1 (Gateway)
- **Laptops**: DHCP assigned addresses (192.168.0.103)
- **Subnet Mask**: 255.255.255.0

### Security
- **Encryption**: WPA2-PSK
- **SSID**: LabWiFi
